# Порівняння ефективності алгоритмів пошуку підрядка

## Використані алгоритми

|  Алгоритм                   | Складність підготовки | Складність пошуку   |
|-----------------------------|-----------------------|---------------------|
| **Боєр-Мур**                |      O(m)             | O(n/m) у найкращому |
| **Кнут-Морріс-Пратт (KMP)** |      O(m)             | O(n + m)            |
| **Рабін-Карп**              |      O(m)             | O(n + m)            |

## Опис тестування

**Текстові файли:**
- `стаття1.txt` - технічна стаття (~17k символів)
- `стаття2.txt` - стаття про алгоритми (~17k символів)

**Підрядки для тестування:**
|   Тип         |  Стаття 1        |  Стаття 2       |
|---------------|------------------|-----------------|
| **Реальний**  | `"виклики"`      | `"експеримент"` |
| **Вигаданий** | `"користуючись"` | `"індекс"`      |

**Методика:**
- 10 повторів для кожного алгоритму
- Середній час виконання за допомогою `timeit`

## Висновки

### Для обох статей:
1. **Boyer-Moore** — абсолютний лідер (**7-12 разів швидше**)
2. **Rabin-Karp** та **KMP** — близькі за продуктивністю

### Переваги Boyer-Moore:

- Ефективні "стрибки" при невідповідностях  
- Таблиця останнього входження символів  
- Найкращий для довгих текстів  
- Працює "справа наліво"  

### Рекомендації:
- **Великі тексти** -> **Boyer-Moore**   
- **Короткі тексти** -> **KMP** (простіший)  
- **Потоковий пошук** -> **Rabin-Karp**
